Text-shadow

The text-shadow property adds a shadow to each letter or symbol that makes the text look as if it is hovering or popping out. They are drawn behind the text but in front of the background and border. The value for a text-shadow property has multiple measurements: a horizontal offset, a vertical set, color, and a potential blur radius.

Right Shadow

The right shadow is created by the first value in the text-shadow element, which is horizontal distance. The shadow will move to the right as many pixels as you designate. If you put a negative sign in front, the shadow will move the opposite direction. In the example below it has been shifted horizontally 7 px and the blue color has been added in so that we can see the black shadow.

All Directions

Shadows that are applied in all directions are created by changing both the horizontal and vertical values. In the example below, the shadow is offset horizontally to the right 15px and vertically down 15px. This aligns the shadow in the bottom right corner of the text.

Glowing Letters

The optional blur radius is what enables letters to look as if they are glowing. The blur option is done by pixels. The higher the pixels the more blurred it will become. Notice in the CSS that it is not altered horizontally or vertically. The blurred radius combined with the color of the shadow will make it look like it is glowing. Note that it is important to have differing colors with the shadow and the text so that you can tell them a part.

See the Pen glowing by JJ Stephens (@josiahstephens) on CodePen.

In the example below, we see multiple shadows lined up to the right, below, and above. When adding multiple shadows you need to separate them by commas. Depending on how many different shadows are stacked on top of each other, it can lead to making it look three dimensional.

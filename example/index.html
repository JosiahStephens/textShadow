Let's Talk About Text Shadows!

Text shadows are made up of 4 values. We need the horizontal distance, the vertical distance, the blur (which is optional), and the color (also optional). Shall we go through each of these values?


Horizontal Distance

The first value in the "text-shadow" element of CSS is the horizontal distance. The CSS basically creates a duplicate line of text and shifts it horizontally to the right however many pixels you specify. In the example below, I shifted the shadow horizontally only, with no vertical shifts, so we can see what it's like.

See the Pen Horizontal Shadow by Emma Gardner (@emmajoi) on CodePen.


Vertical Distance

This is the second value in the "text-shadow" element. The vertical distance puts the duplicate text, or shadow, below it. The shadow is also shifted by pixels, the same way the horizontal value was shifted. Below is an example of a vertical shift, without the horizontal.

See the Pen Vertical Shadow by Emma Gardner (@emmajoi) on CodePen.


Blur

This element is optional but in my opinion it really makes a shadow look like a shadow! The blur is also done pixels. The higher the number, the more blurry it becomes.

See the Pen Blur Shadow by Emma Gardner (@emmajoi) on CodePen.


Color

This is another optional tag; however, if you know how to use it, you can make your shadows look much better. For this example, I used really heinous colors together but it shows the contrast better. By default, the shadow will be the color of the text.

See the Pen Blur Shadow by Emma Gardner (@emmajoi) on CodePen.


Other Fancy Shadow Options

Flipped Shadow

If you need to have your shadows go the opposite direction, you can use negative numbers as tags!

See the Pen Flipped Shadow by Emma Gardner (@emmajoi) on CodePen.

Stacked Shadow

This is my person favorite style of shadow. If you add more than one shadow, add more by separating with commas, it gives it a block effect. This example, I layered 7 shadows on top of one another to give that effect.

See the Pen Blur Shadow by Emma Gardner (@emmajoi) on CodePen.
